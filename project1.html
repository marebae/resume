AesthetiQR

Project Description

AesthetiQR is an AI-powered QR code generator designed to create visually appealing QR codes based on user prompts. The project combines creativity, usability, and modern web technologies.

The application was built using React for the front-end and MongoDB for managing application data.

Skills Learned

  • React for front-end development
  • MongoDB for database storage and management
  • JavaScript application logic
  • Using Git for Version Control
  • Team collaboration
