Project Description

AesthetiQR is an AI-powered QR code generator designed to create visually appealing QR codes based on user prompts. The project lets users create custom QR codes with their choice of font, color, and pattern. Users can also view previous creations and explore QR codes made by others.

The application was built using React for the front-end and MongoDB for managing application data.
