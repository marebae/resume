AesthetiQR

Project Description

AesthetiQR is an AI-powered QR code generator designed to create visually appealing QR codes based on user prompts. The project lets you make a custom QR code with your choice of font, color, and pattern. We also give users the option to view previous creations and allow users to see other users' QR code creations. The application was built using React for the front-end and MongoDB for managing application data.

Skills Learned

  • MongoDB for database storage and management
  • JavaScript application logic
  • Using Git for Version Control
